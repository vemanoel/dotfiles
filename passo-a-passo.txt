1.
	Se o Unitv estiver instalado no firetv, desinstale


2.
	No FireTV, vai na loja de apps e baixa os aplicativos: X-plore e "send Files to TV"


3.
	No celular, vai na loja de apps e instala o app "Send Files to TV"


4.
	No FireTV, abra o aplicativo X-plore, aceite todas as permissões


5.
	No FireTV, abra o aplicativo X-plore, vá até Configurações e marque a opção "Exibir arquivos ocultos"


6.
	No FireTV, Abra o aplicativo X-plore, apague a pasta "Alarms" se ela existir


7.
	No FireTV, Abra o aplicativo X-plore, abra a pasta "Android", se dentro dela existir um arquivo chamado .config apague ele


8.
	No FireTV, abra o aplicativo X-plore, abra a pasta "Armazenamento interno compartilhado", se dentro dela existir um arquivo chamado .config apague ele


9.
	No FireTV, abra o aplicativo X-plore, abra a pasta "Armazenamento interno compartilhado", se dentro dela existir um arquivo chamado .properties apague ele


10.
	No FireTV, abra o aplicativo "send Files to tv" e clique em "receive"


11.
	No celular, abra o aplicativo "Send Files To TV", clique em "Send", selecione alguma config e envie para o firetv


12. 
	No FireTV, abra o aplicativo X-plore

	Navegue para o lado direito da interface

	Mova o cursor para cima da pasta "Android" e pressione o botão "confirmar/ok"

	Navegue para o lado esquerdo da interface

	Procure pela config enviada anteriormente e mova o cursor para cima dela

	Aperte "confirmar/ok" e segure por alguns segundos, aparecerá um menu, clique em copiar e confirme


13.
	 No firetv, abra o aplicativo X-plore, entre na pasta Android e verifique se a config foi enviada para lá


14.
	No FireTV, abra o aplicativo X-plore, abra a a pasta Android, encontre a config, mova o cursor para cima dela, aperte "confirmar/ok" por alguns segundos, aparecerá um menu, clique em remoar e renomeie para .config


15.
	No FireTV, abra o app x-plore, encontre o arquivo "Unitv free 5.1.0" e instale o apk. Depois abra ele


16.
	Vai pedir para atualizar, mas NÃO ATUALIZE AINDA, ao invés disso, volte para o app x-plore, procure pelo arquivo unitvfree 5.3.1 e instale o apk. Depois abra o Unitv novamente


17.
	 Atualize o Unitv até parar de pedir atualização
